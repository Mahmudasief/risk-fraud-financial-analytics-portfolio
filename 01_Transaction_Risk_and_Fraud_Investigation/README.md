# Case 1: Transaction Risk & Fraud Investigation

This case demonstrates transaction monitoring and alert review using judgment-based escalation vs. clearance decisions, with an explicit focus on reducing false positives while maintaining risk coverage.

### Case Scope
This case simulates transaction monitoring and alert review for a business checking account, focusing on identifying risk patterns, differentiating false positives, and applying judgment-based escalation decisions.

### Artifacts
- **Raw Transactions:** Synthetic wire transaction dataset used for alert review
- **Analyst Workpapers:** Excel-based alert review with risk flags, decisions, and rationale
- **Investigation Summary:** One-page written disposition documenting findings and escalation logic

### Skills Demonstrated
- Transaction monitoring and alert triage
- Risk indicator identification (velocity, structuring, pass-through, offshore layering, jurisdictional risk)
- Judgment-based escalation vs. clearance decisions
- False positive reduction
- Professional investigation documentation
